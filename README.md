Dados de utilização de telefones móveis
Este repositório contém uma análise exploratória sobre o dataset de padrões de consumo móvel. O objetivo é identificar tendências, padrões e insights relevantes sobre o uso de telefones celulares.

Descrição
O projeto analisa dados relacionados à utilização de telefones móveis, abordando aspectos como frequência de uso, tipos de serviços utilizados, perfil dos consumidores, entre outros. A análise foi realizada utilizando Python e bibliotecas populares para manipulação e visualização de dados.

Principais Resultados
Identificação dos principais padrões de consumo de serviços móveis.
Visualizações que facilitam a interpretação dos dados.
Insights para possíveis ações de negócio ou pesquisa futura.
